# Document Heading

Exam Galaxy

Ultra-minimal Next.js site for downloadable textbooks and exam resources.

Quick start

- Install: `pnpm install`
- Dev: `pnpm dev`
- Build: `pnpm build`
- start: `pnpm start`

Structure

- `app/` — routes & pages
- `components/` — UI components
- `lib/` — data/helpers

That's it.
